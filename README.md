# 🏙 Smart City AI Operations Center

AI-powered real-time traffic surveillance and incident intelligence system.

## 🚀 Features

- Real-time vehicle detection & tracking
- Collision and stopped vehicle detection
- AI-powered forensic incident reports
- Zone heatmap analytics
- Risk scoring engine
- Live WebSocket dashboard
- Predictive escalation modeling
- Replay GIF generation
- City intelligence API
- Docker-ready deployment

## 🧠 Tech Stack

- FastAPI
- PyTorch
- OpenCV
- YOLO
- LSTM
- WebSocket
- Chart.js
- Leaflet.js
- Groq LLM
- Docker
